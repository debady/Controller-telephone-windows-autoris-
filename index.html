Introduction

Ce guide fournit des instructions détaillées pour afficher l'écran de votre téléphone Android sur votre ordinateur Windows. Plusieurs méthodes sont présentées, ainsi que des solutions aux problèmes courants.

Méthode 1 : Utiliser l'Application Officielle Windows "Téléphone Link"

Prérequis

  • Téléphone Android avec Android 7.0 ou une version ultérieure.
  • Ordinateur Windows exécutant Windows 10 ou 11.
  • Les deux appareils connectés au même réseau Wi-Fi.

Étapes

  1. Sur le PC : Ouvrez l'application "Téléphone Link" (anciennement appelée "Votre Téléphone"). Si elle n'est pas installée, téléchargez-la depuis le Microsoft Store.
  2. Sur le Téléphone : Téléchargez l'application "Link to Windows" depuis le Google Play Store et connectez-vous avec votre compte Microsoft.
  3. Connexion des Appareils : Suivez les instructions pour appairer les deux appareils via le même réseau Wi-Fi.
  4. Affichage de l'Écran : Une fois connecté, vous pourrez projeter l'écran de votre téléphone sur votre PC et interagir avec vos applications.

Méthode 2 : Utiliser "Scrcpy" (Gratuit et Open Source)

Prérequis

  • Téléphone Android avec le Mode Développeur et le Débogage USB activés.
  • Ordinateur Windows avec les Android Platform Tools installés.

Étapes

  1. Télécharger et Installer Scrcpy : Téléchargez depuis le GitHub officiel de Scrcpy et extrayez les fichiers dans un dossier accessible.
  2. Activer le Mode Développeur sur le Téléphone : Allez dans Paramètres > À propos du téléphone > Numéro de build et appuyez 7 fois sur le numéro de build pour activer les options pour les développeurs.
  3. Activer le Débogage USB : Allez dans Paramètres > Options pour les développeurs et activez Débogage USB.
  4. Connecter le Téléphone à l'Ordinateur : Branchez le téléphone via un câble USB et autorisez le débogage USB sur votre téléphone si une invite apparaît.
  5. Lancer Scrcpy : Double-cliquez sur scrcpy.exe dans le dossier extrait. L'écran de votre téléphone devrait s'afficher sur votre ordinateur.

Utilisation de Scrcpy en Mode Sans Fil

  1. Configurer le Mode TCP/IP : Connectez votre téléphone via USB, ouvrez l'invite de commandes dans le dossier de Scrcpy et exécutez : adb tcpip 5555
  2. Trouver l'Adresse IP du Téléphone : Allez dans Paramètres > Wi-Fi > [Nom du réseau] > Détails et notez l'adresse IP (par exemple, 192.168.1.2).
  3. Connecter via Wi-Fi : Déconnectez le câble USB, puis tapez dans l'invite de commandes : adb connect 192.168.1.2:5555. Si la connexion est réussie, lancez scrcpy comme d'habitude.

Méthode 3 : Utiliser "ApowerMirror" (Version Gratuite avec Filigrane)

Étapes

  1. Télécharger ApowerMirror : Téléchargez l'application sur le site officiel d'ApowerMirror pour le PC et installez l'application depuis le Google Play Store sur le téléphone.
  2. Projeter l'Écran via Wi-Fi : Assurez-vous que les deux appareils sont sur le même réseau Wi-Fi. Ouvrez l'application sur les deux appareils, appuyez sur "Miroir" sur le téléphone et sélectionnez votre PC.
  3. Projeter l'Écran via USB (Optionnel) : Connectez le téléphone au PC avec un câble USB et activez le débogage USB.

Méthode 4 : Utiliser "Vysor" (Version Gratuite Limitée)

Étapes

  1. Télécharger Vysor : Téléchargez depuis le site officiel de Vysor sur le PC et installez l'application sur le téléphone via le Google Play Store.
  2. Connecter le Téléphone au PC : Activez le débogage USB et branchez le téléphone via un câble USB.
  3. Lancer Vysor : Ouvrez Vysor sur le PC et cliquez sur "View" pour afficher l'écran du téléphone.

Résolution des Problèmes Communs

Problème : Le Téléphone se Déconnecte Lorsque le Débogage USB est Désactivé

Cause : Le débogage USB est nécessaire pour maintenir la connexion entre le téléphone et le PC lors de l'utilisation de certaines applications.

Solutions :

  1. Utiliser le Mode Sans Fil Sans Débogage USB : Avec des applications comme ApowerMirror, vous pouvez projeter l'écran via Wi-Fi sans le débogage USB.
  2. Maintenir le Débogage USB Activé : Cochez "Toujours autoriser depuis cet ordinateur" lorsque l'invite apparaît.
  3. Utiliser des Alternatives : Des applications comme Link to Windows, AirDroid, ou TeamViewer QuickSupport permettent un contrôle à distance sans débogage USB.

Conseils Supplémentaires

  • Utiliser une Connexion Wi-Fi Stable : Pour éviter les interruptions lors de la projection sans fil.
  • Tester Différentes Méthodes : Ce qui fonctionne le mieux peut varier en fonction de votre appareil et de vos besoins.
  • Exploiter les Fonctionnalités Avancées : Des outils comme Scrcpy et ApowerMirror offrent des options comme l'enregistrement d'écran et le contrôle via clavier/souris.

Conclusion

Projeter l'écran de votre téléphone Android sur votre PC Windows peut être réalisé de plusieurs façons. Que vous préfériez utiliser des outils officiels comme Téléphone Link ou des applications tierces, ce guide vous a fourni des instructions détaillées pour chaque méthode.
